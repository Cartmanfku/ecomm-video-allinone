# SUB_SKILL: Grid Prompt

根据分镜描述生成多宫格生图指令，输出本技能可直接消费的 compact JSON。支持 4/6/9/12/16/25 格自适应，含素材清单和 CG 微观面板质感指南。

## 下游对接

本 Skill 输出的 `image_gen_input` JSON 可直接作为本技能图片脚本的 `submit.py` 输入：

```bash
python scripts/image/submit.py '<image_gen_input的compact JSON>'
```

model 字段对应关系：`nano2`（MiniDesk 2，默认）、`nano-pro`（MiniDesk Pro，高质量）、`seedream`（Seedream V5.0 Lite，最多14张参考图）。

## 输入

```json
{
  "sub_shots": [
    { "panel": "P1", "subject": "办公桌", "action": "键盘文件杂乱", "purpose": "Hook" },
    { "panel": "P2", "subject": "女生", "action": "伸手够杯子", "purpose": "痛点" }
  ],
  "product": "保温杯",
  "style": "写实商业摄影",
  "grid_size": "auto",
  "product_images": [],
  "has_human_face": true
}
```

也可以直接用自然语言描述分镜，Skill 会自动解析。

| 字段 | 默认值 | 说明 |
|---|---|---|
| `sub_shots` | 必填 | 子分镜列表，可以是结构化 JSON 或自然语言 |
| `product` | 从描述推断 | 产品名称 |
| `style` | `写实商业摄影` | 画风 |
| `grid_size` | `auto` | 宫格尺寸，auto 按子分镜数自动选择 |
| `product_images` | `[]` | 产品参考图，用于 @Tag 一致性锚定 |
| `has_human_face` | `false` | 是否含人脸，决定是否加渔网袜指令 |

## 宫格尺寸自适应

| 子分镜数 | 推荐宫格 | 布局 | 适用场景 |
|---|---|---|---|
| 3-4 | 4 格 | 2x2 | 极短视频（5-8s） |
| 5-6 | 6 格 | 2x3 | 短视频（10s） |
| 7-9 | 9 格 | 3x3 | 标准 15s 视频 |
| 10-12 | 12 格 | 3x4 | 详细 15-20s |
| 13-16 | 16 格 | 4x4 | 长视频（20-30s） |
| 17-25 | 25 格 | 5x5 | 30s+ 完整广告 |

决策优先级：
1. 用户手动指定 `grid_size` → 直接使用
2. 按子分镜数匹配上表
3. **宁少不多。** 5 个子分镜用 6 格比用 9 格好

## 叙事节奏布局

每种宫格尺寸有对应的叙事节奏分配：

**4 格（2x2）**
```
P1 P2    开场 + 核心动作
P3 P4    结果 + 收束
```

**6 格（2x3）**
```
P1 P2 P3    开场 + 痛点 + 产品出场
P4 P5 P6    核心动作 + 效果 + 收束
```

**9 格（3x3）**
```
P1 P2 P3    开场段（Hook + 痛点/引入）
P4 P5 P6    中段（核心动作/卖点证明）
P7 P8 P9    收束段（结果/品牌/CTA）
```

**12 格（3x4）**
```
P1  P2  P3     开场段
P4  P5  P6     功能段 A
P7  P8  P9     功能段 B
P10 P11 P12    收束段
```

**16 格（4x4）**
```
P1  P2  P3  P4     开场段
P5  P6  P7  P8     功能段 A
P9  P10 P11 P12    功能段 B
P13 P14 P15 P16    收束段
```

**25 格（5x5）**
```
P1-P5      开场段
P6-P10     功能段 A
P11-P15    功能段 B
P16-P20    效果/证言段
P21-P25    收束段
```

每格只保留 4 类信息：
1. 镜头目标（这一格要证明什么）
2. 视觉主体（画面中心是什么）
3. 动作变化（主体在做什么动作）
4. 情绪节奏（快/稳/停留）

## 导演公式

每格按 `[主体] + [动作] + [场景]` 描述，整体用一段话串联。

### 通用模板

```
以{product}为主体，生成一个{rows}x{cols}的宫格图片，展示以下{panel_count}个连续分镜，
画风统一为{style}，比例16:9：

P1: {subject}在{scene}中{action}，{visual_detail}
P2: {subject}{action}，{visual_detail}
...
P{N}: {product}收束展示，{brand_detail}
```

### 含人脸时的渔网袜模板

当 `has_human_face=true` 时，自动使用此模板：

```
以{product}为主体，生成一个{NxM}的宫格图片，展示以下{总格数}个连续分镜，
画风统一为写实商业摄影，比例16:9。
凡出现人物面部的分镜，保持人物表情发型，面部覆盖黑色细款渔网袜，网眼均匀，不遮挡轮廓，
身体、服装、头发和产品不做任何覆盖处理：

P1: {主体} + {动作} + {场景特征}
P2: ...
P3: 年轻女性手持洗发水微笑，面部覆盖黑色细款渔网袜网眼均匀，身体及产品清晰写实
...
P{N}: {收束画面}
```

渔网袜写法要点：
- **必须写"黑色细款渔网袜"**，生图模型对此描述识别最稳定
- **必须写"网眼均匀，不遮挡轮廓"**，确保网格不盖住五官
- **身体/服装/头发/产品不加覆盖**，只处理面部

哪些子格需要加渔网袜：

| 画面内容 | 被拦概率 | 是否加 |
|---|---|---|
| 正面大头照、眼神直视 | 高 | 必须加 |
| 半身照、自然表情 | 中高 | 必须加 |
| 3/4 侧脸 | 中 | 加 |
| 侧脸 profile | 低 | 可不加 |
| 远景全身（脸部像素小） | 很低 | 不需要 |
| 背影 / 手部 / 产品特写 | 不会拦 | 不需要 |

控制在 30-60 字/格，总长按格数等比缩放。

## CG 微观面板质感指南

当子分镜需要展示头皮、毛囊、角质层、分子结构等 CG 微观画面时，**禁止**使用"CG医学微观"这种笼统前缀（会生成黏土/塑料质感）。

必须遵循的 CG prompt 写法：

| 维度 | 必须包含 | 禁止写法 |
|---|---|---|
| **成像参考** | "扫描电子显微镜超微距特写" / "高端药妆广告级3D渲染" | "CG医学微观" / "CG微观" |
| **组织质感** | "湿润粉色组织表面" / "细密毛孔与微血管纹路" / "半透明角质层" | 无质感描述 |
| **景深与光照** | "浅景深虚化背景" / "侧逆光" / "次表面散射" | 平光 / 无光照描述 |
| **渲染级别** | "Unreal Engine 5 电影级渲染" / "8K超高清" | 无渲染级别标注 |
| **色调** | "蓝青色环境光 + 暖色组织本色" / "冷暖对比" | "蓝色冷调背景"单独使用 |

CG 面板 prompt 模板：

```
扫描电子显微镜超微距特写，浅景深，{具体组织描述}，
{具体动作/状态}，
{光照描述}，
Unreal Engine 5 电影级渲染，8K
```

对比示例：

| 改前（假） | 改后（真实感） |
|---|---|
| CG医学微观，头皮角质屏障层被化学粒子击穿 | 扫描电子显微镜超微距特写，浅景深，湿润粉色头皮组织表面覆盖半透明角质鳞片层，一组橙红色结晶粒子穿透角质层，穿透处鳞片碎裂翻卷露出粉红真皮层，侧逆光穿透碎裂缝隙产生明亮边缘光，UE5 电影级渲染，8K |

**让 CG 画面像"高端药妆广告里的电子显微镜特写"，不是"中学生物课本上的示意图"。**

## 素材清单

每次输出 grid_prompt 时同时生成素材清单。

### 全局素材表

跨单元重复使用的素材统一编号：

```
| 编号 | 素材名称          | 用途       | 来源     | 使用单元  |
|------|-------------------|-----------|---------|----------|
| G1   | 创始人半身照       | 人物参考   | 客户提供 | 1,2,3    |
| G2   | 产品瓶身正面高清图 | 产品锚定   | 客户提供 | 全部      |
```

### 单元素材清单

每个单元在 grid_prompt 之前列出需要的具体素材：

```
| 素材            | 全局编号 | 对应面板   | 备注         |
|-----------------|---------|-----------|-------------|
| 创始人半身照     | G1      | P1,P3,P6 | 正面/侧面    |
| 产品瓶身图       | G2      | P5,P9    | 产品特写      |
```

## 输出

```json
{
  "grid_size": "3x3",
  "panel_count": 9,
  "grid_prompt": "以保温杯为主体，生成一个3x3的九宫格图片...",
  "image_gen_input": {
    "prompt": "以保温杯为主体，生成一个3x3的九宫格图片，展示以下9个连续分镜，画风统一为写实商业摄影风格，比例16:9：P1: 办公桌忙碌开工...",
    "model": "nano2",
    "aspect_ratio": "16:9",
    "resolution": "1K"
  },
  "input_materials": [
    { "name": "产品正面图", "global_id": "G1", "panels": ["P3","P5","P9"] }
  ]
}
```

`image_gen_input` 是 compact JSON，可直接传给 `scripts/image/submit.py`。当有产品参考图时，增加 `reference_images` 字段：

```json
{
  "image_gen_input": {
    "prompt": "...",
    "model": "nano2",
    "aspect_ratio": "16:9",
    "resolution": "1K",
    "reference_images": ["media/product.jpg"]
  }
}
```

## 提示词硬约束

### 必须做
- 用物理语言："灰尘猛烈扬起"、"杯盖弹开"、"手指按压按钮"
- 每格都有可拍摄动作
- 镜头术语明确：推近/平移/俯拍/定格/横移/拉远
- 用 `reference_images` 做产品一致性锚定

### 禁止做
- 禁止情绪词："震撼"、"高级感"、"氛围感"、"史诗"
- 禁止导演口吻："此处观众应该感受到..."
- 禁止 S1/S2/S3 标签出现在最终画面
- 禁止大段字幕或文字墙
