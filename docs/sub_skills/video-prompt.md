# SUB_SKILL: Video Prompt

根据宫格图和分镜信息，生成逐秒视频描述，输出本技能可直接消费的 compact JSON。

## 下游对接

本 Skill 输出的 `video_gen_input` JSON 可直接作为本技能视频脚本的 `submit.py` 输入：

```bash
python scripts/video/submit.py '<video_gen_input的compact JSON>'
```

model 字段对应关系：`fast`（Seedance 2.0 Fast，默认）、`pro`（Seedance 2.0，高质量）、`1.5-pro`（Seedance 1.5 Pro，支持1080p）。

## 核心理念

告诉 AI **你要什么** 和 **感觉如何**，而不是控制每个像素。
最佳 prompt 长度 30-100 词，中文更好。
I2V 提示词只写运动，不重复图片已包含的静态元素。

## 输入

```json
{
  "sub_shots": [
    { "panel": "P1", "time_range": "0-2s", "subject": "办公桌", "action": "忙碌开工" },
    { "panel": "P2", "time_range": "2-4s", "subject": "女生", "action": "伸手够杯子" }
  ],
  "duration": 15,
  "product_category": "水杯",
  "scene_context": "办公室",
  "style": "写实商业摄影",
  "platform": "douyin",
  "clean_screen_level": "L1",
  "has_human_face": true
}
```

| 字段 | 默认值 | 说明 |
|---|---|---|
| `sub_shots` | 必填 | 子分镜列表，含时间分配 |
| `duration` | 必填 | 视频总时长（秒） |
| `product_category` | 从描述推断 | 品类名称 |
| `scene_context` | 从描述推断 | 场景描述 |
| `style` | `写实商业摄影` | 画风 |
| `platform` | `douyin` | 目标平台 |
| `clean_screen_level` | `L1` | 净画面等级 |
| `has_human_face` | `false` | 宫格图中是否含渔网袜处理的人脸 |

## 视频描述模板

```
这是{product_category}（{scene_context}）的{duration}秒电商广告，{style}。
使用参考图作为分镜参考，按以下时间顺序逐帧全屏播放：

0-Xs: {镜头动作}，{主体状态}，{建立什么}
X-Ys: {镜头动作}，{主体动作}，{过渡关系}
...
最后Ns: {收束画面}，{品牌/CTA}

净画面，{净画面规则}。{平台补丁}。
```

### 含人脸时的附加指令

当 `has_human_face=true` 时，在视频描述末尾自动追加：

```
参考图中人物面部的网状覆盖物仅为参考标记，视频中人物面部必须干净清晰，皮肤光滑自然，无任何网格、网纱、遮挡物。
```

并在每个含人脸的时间段追加"面部干净无遮挡"描述。

## 逐秒描述写法

每秒 1-2 句，必须包含：
1. **镜头动作**（推近/平移/俯拍/定格/横移/拉远）
2. **主体状态**（在做什么可拍摄动作）
3. **与上一秒的衔接关系**（平滑过渡/切换/停留延续）

### 不同时长的节奏建议

| 时长 | 宫格数 | 每格平均时长 | 节奏特征 |
|---|---|---|---|
| 5s | 4 格 | ~1.2s | 快闪，每格一个关键帧 |
| 10s | 6 格 | ~1.7s | 紧凑，前 2 格快后 4 格正常 |
| 15s | 9 格 | ~1.7s | 标准，3-3-3 节奏 |
| 20s | 12 格 | ~1.7s | 稳定，可有适当停留 |
| 30s | 16-25 格 | ~1.5-1.9s | 完整叙事，允许节奏起伏 |

### 关键约束
- 时间段必须覆盖全时长，无重叠无空洞
- 每段必须有可执行动作词
- I2V 提示词只写运动和节奏，不重复图片中已有的静态元素
- 通过 `image_url` 字段引用宫格图做全局参考

## 净画面等级

| 等级 | 规则 | prompt 写法 |
|---|---|---|
| `L0` | 完全无文字 | `净画面，无字幕，无文字水印，无任何文字` |
| `L1` | 允许极短标签（<=4字，<=2处） | `净画面，无字幕，无文字水印。仅允许极简短标签（<=4字，<=2处）` |
| `L2` | 允许说明性文字（科普场景） | `净画面，无大段字幕。允许必要科普短术语` |

## 平台补丁

默认按抖音快节奏构建。如果投放平台不同，微调以下 3 项（不改主叙事）：

| 平台 | 节奏 | 文案密度 | CTA 语气 | 补丁文本 |
|---|---|---|---|---|
| **抖音（默认）** | **快（1.5-2s/镜）** | **高信息密度** | **中等** | `镜头节奏紧凑，信息密度高，3秒内建立关注点` |
| TikTok | 快 | UGC 真实感 | 弱 | `节奏快速，UGC质感，前2秒强Hook` |
| 小红书 | 稳（2-3s/镜） | 种草自然 | 弱 | `节奏舒缓自然，种草质感，避免硬广感` |
| YouTube | 稳（2-3s/镜） | 清晰讲解 | 中等 | `镜头稳定，讲解清晰，结构完整` |

## 提示词硬约束

### 必须做
- 用物理语言："灰尘猛烈扬起"、"杯盖弹开"、"手指按压按钮"
- 每秒都有可拍摄动作
- 镜头术语明确：推近/平移/俯拍/定格/横移/拉远
- I2V 只写运动，不重复写静态元素（图片已包含）
- 通过 `image_url` 做角色/产品一致性锚定

### 禁止做
- 禁止情绪词："震撼"、"高级感"、"氛围感"、"史诗"
- 禁止导演口吻："此处观众应该感受到..."
- 禁止 S1/S2/S3 标签出现在最终画面
- 禁止大段字幕或文字墙
- 禁止把宫格整图直接作为视频画面播放

## 输出

```json
{
  "video_prompt": "这是大容量保温杯（办公室场景）的15秒电商广告...",
  "video_gen_input": {
    "prompt": "这是大容量保温杯（办公室场景）的15秒电商广告，写实商业摄影。使用参考图作为分镜参考...",
    "model": "fast",
    "aspect_ratio": "16:9",
    "duration": 15,
    "image_url": "media/<宫格图路径>"
  }
}
```

`video_gen_input` 是 compact JSON，可直接传给 `scripts/video/submit.py`。`image_url` 指向上一步 `scripts/image/download.py` 下载的宫格图本地路径。

## 完整示例

### 示例：保温杯 15s（抖音）

video_prompt 文本：
```
这是大容量保温杯（办公室场景）的15秒电商广告，写实商业摄影。
使用参考图作为分镜参考，按以下时间顺序逐帧全屏播放：

0-2s: 办公桌面中景，键盘文件杂乱，镜头缓慢横移扫过桌面，建立忙碌氛围
2-3s: 女生右手伸向角落普通杯子，手指刚触到杯壁，工作节奏被打断
3-5s: 切换到保温杯特写，手持保温杯放到饮水机下，水流稳定注入，突出大容量
5-6s: 杯盖特写，手指按压按钮，杯盖弹开的瞬间动作
6-8s: 中近景，女生端起保温杯饮水，嘴唇触杯口，表情放松
8-10s: 镜头推近到杯身材质特写，316不锈钢内壁光泽清晰可见
10-12s: 侧面中景，热气从杯口缓缓上升，强调保温效果
12-14s: 拉远到桌面全景，女生专注办公，保温杯安静放在右手边
14-15s: 定格多色保温杯并排展示（白/黑/粉/蓝），品牌logo清晰

净画面，无字幕，无文字水印。仅允许极简短标签（<=4字，<=2处）。
镜头节奏紧凑，信息密度高，3秒内建立关注点。
参考图中人物面部的网状覆盖物仅为参考标记，视频中人物面部必须干净清晰，皮肤光滑自然，无任何网格、网纱、遮挡物。
```
